# WPU Plugin Creator

Quickly Create a WordPress Plugin

## How to install on your machine

Go to your favorite tools folder :

```
git clone https://github.com/WordPressUtilities/WPUPluginCreator

```
Add CLI shortcut :

```
. WPUPluginCreator/inc/installer.sh;
```
